HOW TO RUN THE AUTOMATION SCRIPT
================================

	
	
	(1)Open Eclipse
	
	From Market Place Search & Install TestNG

	(2)Go to file => Import
	
	(3)Select General and Select Existing Project Into Workspace

	(4)Select Root Directory And Browse The Project Location

	(5)Click Finish

	(6)After Import Of The Project User Should Change The Chrome Driver Location Under The @beforeClass annotation
	
	(7)After Import Of The Project User Should Change The Chrome Driver Locattion Under The @test annotation 

	(8)Run The Project ==> RUN AS TESTNG
